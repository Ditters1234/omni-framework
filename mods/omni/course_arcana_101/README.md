# Arcana 101 Course

This addon expands the starter course into a small assignment chain.

## What This Teaches

- `data/activities.json`: adding scheduled course activities that share the base `arcana_101_grade` progress stat.
- `data/quests.json`: using activity-history objectives, stat objectives, and last-outcome objectives to build a course assignment.
- `data/entities.json`: adding a TA NPC with a normal backend interaction that starts the addon quest.

## Play Pattern

Talk to Roan in the Lecture Hall to start the Stabilization Assignment. Attend the problem seminar, complete archive practice, and finish the practical exam. Strong intelligence and willpower produce better grade progress, while weaker builds can still advance through fallback outcomes.

## Assets

This addon includes local placeholder assets under `assets/`. They are copied from current Vanguard base placeholders while final course art and audio are still in progress.
